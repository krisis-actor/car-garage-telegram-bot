# "Гараж"
## Цели проекта
### 1. Создание «вечного» цифрового двойника автомобиля
Хранилище всей технической истории: общий пробег, пробег с момента последнего ТО, пробег с момента замены какой-либо детали. Данные переживут любой сбой бортового компьютера или замену аккумулятора.

### 2. Формирование персональной базы знаний о запчастях и материалах
Каждый пользователь получает возможность создавать и вести свой каталог всех приобретаемых для автомобиля деталей и расходников. К каждой позиции можно прикрепить прямые ссылки для покупки с любых маркетплейсов и магазинов, а также заметки о качестве товара.

### 3. Создание открытой библиотеки экспертных инструкций по обслуживанию
Аккредитованные механики создают пошаговые руководства для конкретных моделей автомобилей. Любой водитель получает к ним доступ, что позволяет в первую очередь ответить на вопрос "Везти ли машину в сервис?". В случае принятия решения о проведении работ самостоятельно - следовать пошаговой инструкции от механика. Если принято решение везти в сервис, то следить за ходом работ/сравнить заказ-наряд с инструкцией.

### 4. Перевод задачи обслуживания автомобиля в фоновый режим
Бот присылает уведомление пользователю о небходимости замены жидкостей или деталей, исходя из информации указанной в каточке авто.
