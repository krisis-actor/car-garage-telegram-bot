Название проекта "Гараж".
Цель проекта: занесение информации касательно личного авто. На основе введенных пользователем данных боту будут доступны 
несложные расчеты, к примеру, срок обновления ОСАГО, замена масла и фильтров, замена тормозных колодок и так далее. Не менее 
важным аспектом владения авто - это знать где и что покупать, поэтому предусмотрена возможность составления реестра 
материалов с артикулами/ссылками на маркетплейсы для конкретного авто. Помимо базовых записей о транспорте, есть возможность 
прикреплять к конкретной модели чек листы/туториалы по самостоятельному обслуживанию. 
Список ролей: Администратор, Водитель, Механик, Автомобиль
Фунции Автомобиля:
  -Регистрация в системе. Введение данных об автомобиле (марка, модель).
  -Первое ТО.
